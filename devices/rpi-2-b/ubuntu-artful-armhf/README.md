Ubuntu is a free and open source software which is developed by [Canonical](https://canonical.com/). `rpi-2-b/ubuntu-xenial-armhf` allows building custom 32-bit Ubuntu [Artful](https://wiki.ubuntu.com/ArtfulAardvark/ReleaseNotes) images for [Raspberry Pi 2 B](https://raspberrypi.org/products/raspberry-pi-2-model-b/) and [Raspberry Pi 3 B](https://raspberrypi.org/products/raspberry-pi-3-model-b/). Note that the kernel package, which the system is based on, is located in the universe area of the official archive. It means that the package is a [community-maintained](https://wiki.ubuntu.com/MOTU/) and not supported by the Ubuntu Security team.
